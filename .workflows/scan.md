# scan

扫描代码，补充缺失的文档注释

1. 扫描 `Scripts/` 目录下的所有 C# 文件（`.cs`）
   - 识别公共类（`public class`）、公共接口（`public interface`）、公共结构体（`public struct`）
   - 识别公共方法（`public` 方法）、公共属性（`public` 属性）、公共字段（`public` 字段）
   - 检查是否已有 XML 文档注释（`/// <summary>`）

2. 对于缺少文档注释的公共成员，根据代码上下文补充：
   - 类/接口/结构体：根据类名、继承关系、实现接口等推断用途，编写简要说明
   - 方法：根据方法名、参数、返回值、方法体逻辑推断功能，编写说明
   - 属性：根据属性名、类型、getter/setter 逻辑推断用途，编写说明
   - 字段：根据字段名、类型、使用场景推断用途，编写说明

3. 文档注释格式：
   - 使用中文编写
   - 使用标准的 XML 文档注释格式（`/// <summary>`）
   - 对于方法，包含参数说明（`<param name="xxx">`）和返回值说明（`<returns>`）
   - 保持简洁明了，重点说明用途和功能

4. 跳过以下情况：
   - 已存在文档注释的成员
   - 私有成员（`private`、`internal`）
   - 自动属性（除非特别重要）
   - 测试相关代码

5. 批量处理：
   - 按文件逐个处理，确保每个文件的文档注释完整
   - 处理完成后检查是否有语法错误

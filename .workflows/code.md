# code

编码任务。遵循项目现有代码风格。

- 命名：PascalCase(类型/方法)，camelCase(字段/变量/参数)
- 注释：只写文档注释(`/// <summary>`)，不写行内注释，使用中文
- 使用现代 C# 特性(`?.` `??` `new()` `var`)，字段优先 `readonly`
- 日志用 `Log.Print/PrintErr/PrintException`
- 不要重命名现有代码
- 完成后无需关心格式化和 lint
- 尽可能不要新增字段或者setter
- 异步的临时状态尽可能放进闭包里作为局部变量而不是字段
- 如果封装的方法只调用一次，那应该倾向于不封装或者闭包
